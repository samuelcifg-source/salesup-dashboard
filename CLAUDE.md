# SalesUp Dashboard

Dashboard interno de SalesUp para gestionar clientes, métricas y operaciones.

## Stack
- **Framework**: Vite + React
- **CSS**: Tailwind CSS
- **Deploy**: `dist/`

## Estructura
- `src/` — Código fuente React
- `dist/` — Build de producción
- `worker/` — Cloudflare Worker (o similar)
- `salesup-dashboard-v2/` — Versión 2 del dashboard
- `WEBSITE BUILDER/` — Herramienta de creación de webs
- `Sales App dashboard v2, copia de seguridad/` — Backup de v2

## Notas
- Responder siempre en español
- El `.env.local` con credenciales de Supabase está en `salesup-dashboard-v2/`
